# Anaphora

**Definition.** The same word or phrase starts each successive clause or sentence.

**Example.**
- Wrong: We will rebuild the index, deprecate the old API, and migrate every customer.
- Good: We will rebuild the index. We will deprecate the old API. We will move every customer ourselves.
