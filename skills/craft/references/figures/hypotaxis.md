# Hypotaxis

**Definition.** Long sentences with nested subordinate clauses — the prose of nuance and qualification.

**Example.**
- Wrong: The vendor is cheap. Their support is poor. Their API is unstable. We should not pick them.
- Good: Although the vendor is cheap, the combination of weak support — which will cost us engineering hours we have already promised elsewhere — and an unstable API — which will cost us a trust we cannot afford to spend — makes them the wrong choice.
