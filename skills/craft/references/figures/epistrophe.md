# Epistrophe

**Definition.** The same word or phrase ends each successive clause or sentence — anaphora's mirror.

**Example.**
- Wrong: The dashboard ships next month, the migration is also planned for then, and the new permissions roll out around the same time.
- Good: The dashboard ships this quarter. The migration ships this quarter. The permissions ship this quarter.
